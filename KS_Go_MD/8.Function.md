# 8.Function 函数
### 函数是基本的代码块，用于执行一个任务。
### Go语言最少有个 main() 函数。
### 可以通过函数来划分不同功能，逻辑上每个函数执行的是制定的任务。
### 函数声明告诉了编译器函3件事：1.函数的名称；2.参数；3.返回类型。

# 函数的声明 Func_Decl (Function Declaration)
### Go语言函数定义格式如下：
```go
func function_name( [parameter list] ) [ return_type ] {
	函数体
}
```
## Go语言允许有函数有以下5种声明：
### 无参数，无返回值，的函数
### 有1个参数，的函数
### 有2个参数，的函数
### 有1个返回值，的函数
### 有多个返回值，的函数
